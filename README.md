# Computer Simulation Studies through MATLAB 💻

This repository documents my learning journey in Computer Simulation during my exchange program at Shibaura Institute of Technology under the guidance of Prof. Takumi Miyoshi and Prof. Taku Yamazaki. It contains practical assignments, experiments, and projects developed throughout the course, focusing on both theoretical understanding and hands-on implementation using MATLAB. My goal is to consolidate knowledge in simulation techniques and demonstrate the application of computational methods to solve real-world problems.

## 📌 About this repository

The materials in this repository follow the structure of the course, starting from basic concepts of computer simulation and progressing toward more advanced topics such as stochastic modeling, queue systems, and network simulations. Each notebook or script reflects a specific class topic and includes implementations, experiments, and analysis. Over time, this repository serves as a structured reference for simulation methods and modeling approaches.

## 📚 Contents

The repository is organized as a sequence of class notebooks. The course begins with an introduction to computer simulation and an overview of the MATLAB environment. It then covers fundamental programming concepts such as flow control structures and user-defined functions.  As the course progresses, it explores data visualization techniques and file operations for handling simulation data. It also introduces the fundamentals of queue theory and its practical applications. In the later stages, the focus shifts to simulating queue systems and wireless network systems. The course concludes with independent simulation projects, where students design, implement, and analyze their own simulation problems.

## ▶️ How to run

The files in this repository can be executed using MATLAB. Make sure you have MATLAB installed, open the scripts or live scripts (.mlx) and run each section step by step to understand the simulation logic. Some simulations may require basic knowledge of MATLAB and numerical methods.

## 📝 Notes

This is a study focused repository. The emphasis is on clarity, incremental learning, and keeping each notebook understandable as a standalone reference.
